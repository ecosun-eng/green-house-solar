# Green House Solar 🌱

Proyecto innovador que utiliza energía solar y tecnología sostenible para fortalecer la agricultura y apoyar el crecimiento del campo campesino.

**"El futuro del campo nace donde la naturaleza y la innovación se unen."**

## Estructura del Proyecto

- 📄 `index.html` - Página "Quiénes Somos"
- 🌍 `ambiente.html` - Página "Medio Ambiente"
- ⚙️ `funcionamiento.html` - Página "Cómo Funciona"
- 🔧 `servicios.html` - Página "Servicios"
- 📞 `contacto.html` - Página "Contacto"
- 🎨 `css/` - Estilos compartidos
- 📦 `assets/` - Imágenes y recursos
